# football-manager-sim

Headless simulation core and benchmark suite for an 11v11 football (soccer)
tactics simulator on a real-size pitch (105×68 m). TypeScript; `src/sim/` is
DOM-free and fully deterministic — a match is a pure function of
(formations, tactics, seed).

This repository is a **compute mirror**: filtered snapshots synced from a
private development repository so that CI benchmarks (regression suite,
tracking/event benchmarks, league meta-audit) run here on GitHub Actions.
Development history and design documents are not published. Not currently
accepting contributions.
